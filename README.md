# Animated Resume Analytics Builder

A full-stack scaffold for a hiring-quality resume builder application.

- `frontend/` — Next.js 14 app router with TypeScript, Tailwind CSS, Recharts, Framer Motion, React Hook Form, and Zod.
- `backend/` — Express.js and TypeScript API backed by MongoDB/Mongoose with clean versioned routing.

## Local development

Start the frontend:

```bash
cd frontend
npm install
npm run dev
```

Start the backend:

```bash
cd backend
npm install
npm run dev
```

The frontend runs at `http://127.0.0.1:3000` and the backend runs at `http://localhost:4000`.
